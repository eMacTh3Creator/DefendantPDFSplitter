Built for bundled court packets.

Designed around the exact review loop that legal and process-service teams repeat every day: detect, correct, split, and package.

Defendant and case detection

Reads PDF text first, falls back to OCR for scanned pages, and looks for common caption, defendant, case, docket, and civil-action labels.

Fast review controls

Accept detected defendant names row by row or fill every blank suggestion at once, then manually correct any edge cases before export.

Case-aware splitting

Consecutive pages with the same defendant stay together only when the case numbers are compatible. Different cases become separate PDFs.

Drop in a PDF. Get named files and a ZIP.

No accounts, no upload step, and no cloud processing. The app works locally with PDFKit, Vision OCR, and a native macOS review table.
